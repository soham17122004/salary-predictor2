# Employee Salary Prediction — Frontend

## Setup
1. `pip install -r requirements.txt`
2. `python app.py`
3. Open http://127.0.0.1:5000

## Deploy on Render
1. Push this project to a GitHub repo (root of repo = this folder's contents).
2. On [render.com](https://render.com), click **New > Web Service** and connect the repo.
3. Set:
   - **Build Command:** `pip install -r requirements.txt`
   - **Start Command:** `gunicorn app:app`
4. Deploy. Render gives you a live `onrender.com` URL once the build finishes.

Note: on Render's free tier, the app spins down after 15 minutes of inactivity and takes ~60 seconds to wake back up on the next request — that's normal, not a bug.

## Structure
- `app.py` — Flask backend, loads the trained model and serves the form
- `templates/index.html` — the frontend form
- `model/` — trained model, scaler, feature names, and dropdown options (generated by train.py)
- `train.py` — retrains the model from the dataset if you need to regenerate the .pkl files
